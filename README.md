# person-Identification-model
A neural network in numpy for person identification. A complete ANN with 2 hidden layers and fully implemented back and forward propagation.
Give number of neurons of your own choice (default : 512->128).


## dependencies:

- numpy
- opencv
- matplotlib.pyplot
- scikit-learn
